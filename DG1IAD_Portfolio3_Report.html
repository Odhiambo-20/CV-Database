3. Security Features and Main Files

  • Authentication: User login and session management in login.php and includes/auth.php.
  • Authorization: Access to CV editing restricted to logged-in owner only in edit_cv.php and includes/auth.php.
  • Password hashing: Passwords hashed with password_hash() and checked by password_verify() in register.php and login.php.
  • Form validation: Server-side validation for required fields, email format, length limits, and password rules in register.php, login.php, edit_cv.php, and includes/helpers.php.
  • SQL injection prevention: PDO prepared statements and bound parameters used in all database queries (mainly index.php, view_cv.php, register.php, login.php, edit_cv.php).
  • XSS prevention: Output escaped using helper function h() in includes/helpers.php and rendered pages.
  • CSRF protection: Token generation and verification in includes/csrf.php; verified for form submissions in register.php, login.php, edit_cv.php, and logout.php.
  • Session/browser hardening: Session cookie flags (HttpOnly, SameSite=Strict, secure on HTTPS) and security headers in includes/bootstrap.php.
